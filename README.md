# 🧺 WhatsApp Chatbot — Med Clean

> Atendimento automatizado via WhatsApp para a lavanderia **Med Clean**, localizada no Guará II — DF.

</div>

---

## 📖 Sobre o Projeto

O **WhatsApp Chatbot Med Clean** foi desenvolvido para modernizar e agilizar o atendimento ao cliente da lavanderia. Com ele, os clientes conseguem obter informações sobre serviços, preços, endereço e contato diretamente pelo WhatsApp — sem precisar esperar por um atendente humano para dúvidas simples.

O sistema também conta com encaminhamento inteligente para atendimento humano, enviando um alerta automático ao responsável quando necessário.

---

## 📌 Funcionalidades

- 📋 **Menu Principal** — Navegação pelos serviços disponíveis
- 💰 **Consulta de Preços** — Tabela de valores dos serviços
- 📞 **Informações de Contato** — Telefone e endereço da unidade
- 👨‍💼 **Atendimento Humano** — Encaminhamento para atendente com alerta automático ao responsável

## 🚀 Como Executar o Projeto

### Pré-requisitos

Antes de começar, certifique-se de ter o [Node.js](https://nodejs.org/) instalado na sua máquina.

### Passo a passo

**1. Clone o repositório**
```bash
git clone https://github.com/seu-usuario/seu-repositorio.git
```

**2. Acesse a pasta do projeto**
```bash
cd nome-do-projeto
```

**3. Instale as dependências**
```bash
npm install
```

**4. Execute o chatbot**
```bash
node index.js
```

---

## 📱 Como Usar

Ao iniciar o sistema:

1. Um **QR Code** será exibido no terminal
2. Abra o WhatsApp no seu celular e acesse **Dispositivos conectados**
3. Escaneie o QR Code com o número desejado
4. ✅ O bot estará **online** e pronto para responder automaticamente
