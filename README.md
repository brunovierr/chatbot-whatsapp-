
WhatsApp Chatbot 🧺

Este projeto consiste em um chatbot de atendimento automatizado via WhatsApp para a lavanderia Med Clean, localizada no Guará II. O objetivo é agilizar a comunicação com os clientes, fornecendo informações essenciais de forma instantânea.
Funcionalidades Atuais

    Menu Principal: Direcionamento para serviços e preços.

    Informações de Contato: Exibição do telefone e endereço da unidade.

    Encaminhamento: Opção para falar diretamente com um atendente humano + alerta enviado ao atendente.

🛠️ Tecnologias Utilizadas

    Node.js: Ambiente de execução.

    JavaScript: Linguagem principal do projeto.

    WhatsApp Web API: Integração com o serviço de mensagens.
